## Setup and Installation

### Prerequisites

Ensure you have the following installed:

- Node.js (>= 16.x)
- npm or yarn
- Git

### Installation Steps

  

First, run the development server:

 ```bash
   npm i
   npm run dev
   #or
   npm start
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
